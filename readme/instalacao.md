# Instalação

Usando o maven

{% hint style="info" %}
Substitua o REPLACE\_VERSION pela versão que desejar
{% endhint %}

```xml
<repositories>
     <repository>
         <id>jitpack.io</id>
         <url>https://jitpack.io</url>
      </repository>
</repositories>
<dependencies>
    <dependency>
	<groupId>com.github.Jhonatas48</groupId>
	<artifactId>mysql_api_simples</artifactId>
	<version>REPLACE_VERSION</version>
     </dependency>
<dependencies>
```
